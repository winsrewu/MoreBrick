# MoreBrick
This is a mod to express my view about the translation issues
about brick and bricks in Chinese in Minecraft, so I won't provide
the English version of readme.

I registered tons of brick of bricks in Minecraft, you are welcome to
reuse them in your own mod or project under the LICENSE of this mod.

这是一个表达我对Minecraft中砖块和砖块的中文翻译看法的模组。

# 声明
个人看法, 并不代表本人赞同或反对Minecraft的中文翻译工作。  
为了区别砖的不同形式, 这里会用砖块代指砖的方块形式和它们对应的方块物品,
砖用于代指用于合成砖块的材料。
碎字前缀表面一块砖块或者是一块砖是破碎的, 与"裂纹"不同。

# 新物品
几乎所有对应砖块的砖, 和砖与砖块的破碎变体。  


# 机制
- 修改了砖块的合成与掉落
- 使用砖打人会让砖破碎
- 被砖或者砖的物品实体形式打到会让实体获得"被砸伤"效果
- 碎砖块有几率掉落, 被它砸到的实体会受伤
- 在碎砖块上有几率受伤
